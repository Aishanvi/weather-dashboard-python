# Weather Dashboard (Python)

A real-time weather application built using Python and OpenWeather API.  
This project fetches live weather details of any city and displays temperature, humidity, and weather conditions through a command-line interface.

## Features

- Fetch live weather data using API
- Display temperature in Celsius
- Show humidity percentage
- Display weather conditions
- Handles invalid city/API errors
- Simple CLI-based interaction

## Technologies Used

- Python
- Requests Library
- OpenWeather API
- JSON

## How to Run

1. Install required package

```bash
pip install requests
```

2. Run the program

```bash
python weather_app.py
```

3. Enter city name when prompted
